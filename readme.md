### Server

1. Add the following variables to the `.env` file

```bash
MONGO_URI
SECRET
```

2. Install the packages

```bash
npm install
```

3. Run the server

```bash
npm run dev
```

### Client

1. Install the packages

```bash
npm install
```

2. Run the client side

```bash
npm run dev
```

3. View the website at `http://localhost:5173`
